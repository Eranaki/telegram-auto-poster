Подсказка

Настройки канала

  • Название — имя канала внутри панели.
  • Chat ID / канал — куда бот будет публиковать посты.
  • Токен бота — токен Telegram-бота для публикаций.
  • Parse mode — формат разметки подписи, чаще всего HTML.
  • Подпись по умолчанию — общий текст подписи для правил канала, если у правила нет собственного шаблона.
  • Отправлять без уведомления — посты публикуются тихо.
  • Защитить контент от пересылки — Telegram попробует ограничить пересылку и сохранение.
  • Канал включен — быстрый переключатель активности канала.

Подсказка

Chat ID / канал

Сюда нужно вписывать адрес, куда бот будет постить.

  • @username_канала — самый удобный вариант для публичных каналов.
  • числовой chat_id — чаще нужен для приватных каналов.

Если нужно получить chat_id приватного канала, добавь бота администратором, опубликуй пост и открой https://api.telegram.org/botТОКЕН/getUpdates.
