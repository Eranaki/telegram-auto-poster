Подсказка

Как настроить правило

  • Источники — папки, из которых правило будет брать файлы. В текущей версии все выбранные источники работают как один общий пул.
  • Интервал — как часто запускается событие публикации.
  • Рандомизация времени — случайное отклонение от интервала в обе стороны.
  • Постов за событие и Интервал между постами — серия публикаций внутри одного запуска правила.
  • Публиковать с часа / до часа — ограничивает публикации указанным окном по времени.
  • Режим выбора файлов:
    Случайно без повторов — берет случайный файл, пока уникальные не закончатся.
    Перемешанный цикл — перемешивает доступные файлы и идет по ним как по циклу.
    По очереди от старых — выбирает файл с самым старым временем публикации.
    Полный случайный выбор — может снова выбрать уже отправленный файл раньше остальных.
  • Переопределить канал — если нужно временно отправлять это правило в другой Telegram-канал.
  • Шаблон подписи — подпись только для этого правила, с приоритетом над подписью канала.
  • Повторять после исчерпания — позволяет снова запускать правило по тем же файлам после полного прохода.
  • Всегда отправлять как документ — полезно, если Telegram плохо обрабатывает медиа как фото или видео.
  • Автоматически конвертировать HEIC в JPEG — для проблемных HEIC/HEIF-файлов перед отправкой как фото.
