Как настроить источник

  • Название — внутреннее имя папки в панели.
  • Путь внутри контейнера — конкретная смонтированная папка внутри /content.
  • Тип файлов — какие типы контента Telegram будет использовать из этой папки.
  • Интервал сканирования — как часто приложение будет искать новые и изменённые файлы.
  • Только ручное сканирование — отключает автоматический запуск сканирования по времени.
  • Сканировать рекурсивно — включает обход вложенных подпапок.
  • Включить источник — быстрый переключатель активности источника.
