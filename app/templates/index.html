Как заполнить канал

  • Название — любое удобное имя канала внутри панели.
  • Chat ID / канал — либо @username_канала, либо числовой chat_id приватного канала.
  • Токен бота — токен Telegram-бота, который будет публиковать посты.
  • Parse mode — формат разметки подписи. Обычно удобно оставить HTML.
  • Подпись по умолчанию — текст, который будет подставляться в посты этого канала, если правило не задаёт свой шаблон.
  • Отправлять без уведомления — публикация без звукового уведомления подписчикам.
  • Защитить контент от пересылки — Telegram попытается ограничить пересылку и сохранение контента.
  • Включить канал сразу — если выключено, правила канала не будут публиковать в него до повторного включения.

Подсказка

Chat ID / канал

Сюда нужно вписывать адрес, куда бот будет постить.

  • @username_канала. Если у канала есть публичный username, это самый удобный вариант. Пример: @my_channel.
  • числовой chat_id. Обычно нужен для приватных каналов или групп, когда username нет. Для каналов это часто отрицательное число, например -1001234567890.

Важно: бот должен быть добавлен в канал и иметь права на публикацию сообщений.

Если у тебя публичный Telegram-канал, почти всегда проще вписать именно @имя_канала.

Как получить chat_id для приватного канала:

  1. Добавь своего бота в канал как администратора.
  2. Опубликуй в канале любое сообщение.
  3. Открой в браузере https://api.telegram.org/botТОКЕН_БОТА/getUpdates, подставив реальный токен.

В ответе ищи блок вроде "chat": { "id": -1001234567890, "type": "channel" }. Вот это id и нужно вписать в поле.

Нюансы: иногда сначала нужно написать боту в личку /start. Если у бота настроен webhook, getUpdates может быть пустым.
