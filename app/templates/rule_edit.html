Подсказка

Как работает правило

  • Источники — все выбранные источники формируют общий пул файлов для правила.
  • Интервал — базовая частота запусков.
  • Рандомизация времени — случайное смещение запуска относительно базового интервала.
  • Постов за событие и Интервал между постами — серия постов подряд внутри одного запуска.
  • Публиковать с часа / до часа — ограничение времени публикации в течение суток.
  • Режим выбора файлов:
    Случайно без повторов — случайный файл, пока не закончатся уникальные.
    Перемешанный цикл — один раз перемешанный проход по пулу.
    По очереди от старых — в первую очередь берет давно не отправлявшиеся файлы.
    Полный случайный выбор — полностью случайный выбор, повторы возможны раньше остальных.
  • Переопределить канал — ручная отправка этого правила в другой канал.
  • Шаблон подписи — подпись только для этого правила.
  • Повторять после исчерпания — после полного прохода правило снова начнет использовать те же файлы.
  • Всегда отправлять как документ — полезно для нестандартных форматов и проблемных медиа.
  • Автоконвертация HEIC — предварительно переводит HEIC/HEIF в JPEG перед отправкой как фото.
