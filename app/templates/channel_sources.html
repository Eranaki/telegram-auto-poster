Подсказка

Как настроить источник

  • Название — внутреннее имя папки в панели.
  • Путь внутри контейнера — смонтированная папка внутри /content.
  • Тип файлов — какие типы контента Telegram будет использовать из этой папки.
  • Интервал сканирования — период автоматического поиска новых файлов.
  • Только ручное сканирование — сканирование будет запускаться только по кнопке.
  • Сканировать рекурсивно — включает вложенные подпапки.
  • Включить источник — делает источник доступным для правил.
